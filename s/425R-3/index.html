§425R-3

Addresses in filings.

⚖ Official version ↗
[§425R-3] Addresses in filings. Whenever a provision of this chapter other than section 425R-10(a)(4) requires that a filing state an address, the filing shall state an actual street address or rural route box number in the State. [L 2009, c 55, pt of §1] Previous Vol08_Ch0401-0429 Next
