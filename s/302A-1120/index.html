§302A-1120

Public library system; board control.

⚖ Official version ↗
[§302A-1120] Public library system; board control. The board, through the state librarian, shall have direct control of the public library system, but not including school libraries. The board may adopt rules under chapter 91 for the purpose of this section. [L 1996, c 89, pt of §2] Previous Vol05_Ch0261-0319 Next
