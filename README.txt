HAWAII REVISED STATUTES — Searchable Local Copy
================================================

HOW TO OPEN
-----------
This is a small website. Because browsers block local file access for
security, it must be served by a tiny local web server (already on your PC
if you have Python).

Easiest way (Windows):
  1. Double-click "Start-HRS.bat".
  2. Your browser opens at http://localhost:8777/
  3. To stop, close the black command window.

Manual way (any OS with Python):
  Open a terminal in this folder and run:
     python -m http.server 8777
  Then visit http://localhost:8777/ in your browser.

USING IT
--------
  • Search box (top): full-text search across every section.
      - "Titles only" limits the search to section headings.
      - The volume dropdown narrows results to one volume.
  • Left sidebar: browse by Volume > Chapter > Section.
  • Each section links back to its official source page on capitol.hawaii.gov.
